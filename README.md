# RandomUser

Android application that loads information from the [Random User Api](https://randomuser.me)

## Technologies:

* Clean Architecture
* SOLID, KISS, DRY
* JUnit
* Retrofit
* Room
* MVVM
* LiveData
* Dagger
* Coil
* Navigation Component
* Night mode
* KSP
